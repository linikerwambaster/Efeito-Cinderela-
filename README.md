# Efeito-Cinderela-
Você já acordou se sentindo 'uó', inchada e com aquela sensação de que a roupa não serve direito? O Efeito Cinderela não é uma dieta restritiva dessas que te fazem passar fome. É um protocolo estratégico de desintoxicação noturna.  Descobrimos que o segredo não é o que você come durante o dia, mas o que você bebe antes de dormir.
